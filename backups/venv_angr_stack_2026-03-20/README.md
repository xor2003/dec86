# Saved angr Stack Patches

This backup captures the local edits that existed in `/home/xor/vextest/venv` before upgrading the active
Python 3.12 angr stack from `9.2.168` to `9.2.205` on 2026-03-20.

## Contents

- `python3.12-angr-stack.tar.gz`
  - Full backup of the active Python 3.12 angr-related site-packages and dist-info metadata before upgrade.
- `python3.10-angr-stack.tar.gz`
  - Full backup of the older Python 3.10 angr-related site-packages and dist-info metadata.
- `patched_files_manifest.json`
  - Files that differed from the wheel `RECORD` hashes at backup time.
- `patched_files/`
  - Raw copies of the patched files as they existed before the upgrade.
- `patches/python3.12-combined.patch`
  - Combined unified diff for the active Python 3.12 edits, suitable as a starting point for reapplying or splitting
    into upstream submissions.
- `patches/python3.12/*.patch`
  - Per-file unified diffs for the active Python 3.12 edits.

## Notes

- The Python 3.12 patch files are generated by comparing the upgraded upstream install against the saved pre-upgrade
  patched files.
- The Python 3.10 files are preserved in `patched_files/python3.10/`, but a meaningful full patch series was not
  regenerated automatically because that environment still contains the already-patched files as its local baseline.
- To inspect the preserved active-stack patches:
  - `patches/python3.12/angr/analyses/calling_convention/utils.py.patch`
  - `patches/python3.12/angr/analyses/typehoon/simple_solver.py.patch`
  - `patches/python3.12/angr/analyses/variable_recovery/variable_recovery_base.py.patch`

## Regeneration

Patch files can be regenerated with:

```bash
venv/bin/python angr_platforms/scripts/export_saved_angr_patches.py \
  --backup-dir /home/xor/vextest/backups/venv_angr_stack_2026-03-20 \
  --site-packages-root /home/xor/vextest/venv/lib/python3.12/site-packages \
  --label python3.12
```
